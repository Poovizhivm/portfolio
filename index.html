Real-Time Indian Sign Language Interpreter

ISL has no everyday translation layer between signers and non-signers. I built a system that reads hand gestures through a webcam and classifies them instantly — trained on a dataset I collected and labelled myself, running on a fine-tuned MobileNetV2 pipeline from pre-processing through live inference.

TensorFlowKerasOpenCVMobileNetV2
94.4% recognition accuracy across six live gesture classes

AI Recipe Finder

Recipe sites are generic — they don't care what's actually in your kitchen. This app connects a local LLM (Llama 3.2, run through Ollama) that parses whatever ingredients you type and generates personalized suggestions on top of a library of 700+ Indian and international dishes, served through a REST API I built from scratch.

Llama 3.2OllamaREST API
80%+ ingredient-to-recipe match accuracy

PDFFlow

Most free PDF tools quietly upload your files to a server. PDFFlow doesn't — every split, merge, compress, and page-numbering operation runs entirely in the browser, so nothing you touch ever leaves your device, even offline.
